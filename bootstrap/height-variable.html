Controlar el height de los elementos

El problema

  • Los elementos internos de una columna col-* no ocupan el 100% del height
  • Por defecto las columnas siempre miden lo mismo de alto
  • Para verificar que esto es así le agregamos un borde a las columnas con .border-danger
  • Queda en evidencia que las columnas tienen la misma altura, pero no su contenido.

El problema entonces no es de las columnas, sino de los elementos internos. En este caso el componente card

Usando flex para ocupar el 100%

  • La clase de Bootstrap .d-flex aplica lógicamente un display flex al elemento seleccionado
  • El efecto "secundario" al aplicar este tipo de display es que los items internos ocupen un 100% del height
  • Esto sucede porque se aplica por defecto la propiedad align-items: stretch
  • Con esta lógica entonces si a todos los elementos col-* le aplicamos la clase d-flex sus elementos internos van a "estirarse" hasta el 100% del height
  • En este caso esto se aplica a los componentes .card
